Para lograr este efecto, debemos entender el concepto de trabajar por capas y utilizamos propiedades CSS de fución de capas (background-blend-mode) y luego propiedades de filtros (filter)

  1. La primer capa que creamos es un background con un patron de líneas repetidas backround: repeating-linear-gradient(45deg, #000, #999, #000 5%)
  2. La segunda capa que creamos, es simplemente un gradiente: background: linear-gradient(45deg, #777, #000);
  3. Luego generamos una fusión entre ambas capas utilizando: background-blend-mode: screen; y obtenemos una fusión entre las dos capas
  4. Importante! En este caso solo tenemos 2 (dos) capas, pero si tuvieramos 3 o más, el orden de las capas si afecta a la fusión, es decir, si fusionamos las capas: 1, 2 y 3... podríamos ver una imagen diferente a si fusionamos las capas: 1, 3 y 2
  5. Finalmente, para obtener una imagen con líneas bien definidas, utilizamos un filtro de CSS... filter: contrast(20)
